# Bitácoras de integración

En esta carpeta se registran los lotes de cambios integrados al repositorio.

Formato de nombre: `YYYY-MM-DD_<tema>.md`

Cada bitácora debe incluir:
- Resumen del lote
- Alcance técnico (archivos/módulos)
- Mapeo al checklist maestro
- Riesgos y deuda técnica/documental
- Acciones y responsables
